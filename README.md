# Pruebas de Software - DAWA

Este proyecto contiene la configuración base para el desarrollo y ejecución de pruebas unitarias automatizadas utilizando **Jest** en un entorno de Node.js.

## Estructura del Entorno
* **Entorno:** Node.js
* **Framework de Pruebas:** Jest
* **Reportes:** Generación automática de cobertura en la carpeta `/coverage`

## Herramientas e Instancias Disponibles
* Script para pruebas unitarias básicas: `npm test`
* Script para pruebas con reporte de cobertura detallado: `npm run test:coverage`
